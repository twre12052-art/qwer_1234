## 🚨 알려진 이슈 및 해결책

### Issue 1: Admin 로그인 시 일반 화면
**해결**: 
```sql
-- Supabase SQL:
UPDATE users SET role = 'admin' WHERE name = 'admin';
```
로그아웃 → 재로그인

### Issue 2: 케이스가 안 보임
**확인**:
- Admin: 모든 케이스 보여야 함
- 일반: 본인 케이스만 보여야 함

### Issue 3: 간병인 동의 후 상태 안 바뀜
**해결**:
- 보호자 화면에서 F5 또는 [새로고침] 버튼
