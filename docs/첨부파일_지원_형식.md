# 📎 첨부파일 지원 형식

## ✅ 지원하는 파일 형식

### 이미지 파일 (사진, 캡처)
- **JPG / JPEG** - 가장 일반적인 사진 형식
- **PNG** - 투명 배경 지원, 캡처에 적합
- **GIF** - 애니메이션 이미지
- **WEBP** - 최신 웹 이미지 형식
- **BMP** - 비트맵 이미지

### 문서 파일
- **PDF** - 스캔본, 문서 (가장 일반적)

### 오피스 문서
- **DOC / DOCX** - Word 문서
- **XLS / XLSX** - Excel 문서
- **PPT / PPTX** - PowerPoint 문서

---

## 📏 파일 크기 제한

- **최대 크기**: 20MB
- 일반적인 업로드 사이트 기준으로 설정

---

## 💡 권장 사용 사례

### 병원 영수증
- ✅ **JPG, PNG** - 스마트폰으로 촬영한 사진
- ✅ **PDF** - 스캔본

### 병원 세부영수증
- ✅ **JPG, PNG** - 사진
- ✅ **PDF** - 스캔본
- ✅ **XLS, XLSX** - 엑셀 파일

### 입퇴원 확인서
- ✅ **JPG, PNG** - 사진, 캡처
- ✅ **PDF** - 스캔본
- ✅ **DOC, DOCX** - 문서 파일

### 간호일지
- ✅ **JPG, PNG** - 사진
- ✅ **PDF** - 스캔본
- ✅ **DOC, DOCX** - 문서 파일

---

## ⚠️ 주의사항

1. **파일명에 확장자가 있어야 함**
   - ✅ `영수증.jpg`
   - ❌ `영수증` (확장자 없음)

2. **파일 크기 확인**
   - 20MB 초과 시 업로드 불가
   - 큰 파일은 압축하거나 분할 필요

3. **파일 형식 확인**
   - 지원하지 않는 형식은 업로드 불가
   - 에러 메시지에 지원 형식 표시됨

---

## 🔍 문제 해결

### "지원하지 않는 파일 형식입니다" 에러
- 파일 확장자 확인
- 지원 형식 목록 확인
- 파일명에 특수문자 제거

### "파일 크기가 너무 큽니다" 에러
- 파일 크기 확인 (20MB 이하)
- 이미지 압축 도구 사용
- PDF 최적화 도구 사용
