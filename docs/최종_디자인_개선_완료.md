## 🧪 테스트 방법

### 1️⃣ 서버 재시작
```bash
Ctrl + C
npm run dev
```

### 2️⃣ 홈페이지 확인
```
1. http://localhost:3000 접속
2. 새로운 디자인 확인:
   - 그라데이션 배경
   - 큰 제목
   - 3개 Feature 카드
   - 3단계 설명
   - CTA 섹션
   - Footer
```

### 3️⃣ PDF 테스트
```
1. 로그인
2. 케이스 생성 + 간병인 동의
3. 케이스 상세 페이지
4. [파일확인 및 다운] 클릭 (헤더)
5. 서류 섹션으로 스크롤 확인 ✅
6. [전체 다운로드 (6페이지)] 클릭
7. PDF 열기:
   - 표지 확인
   - 목차 확인
   - 4종 서류 확인
   - 글자 크기 확인 (큼!)
   - 중앙 정렬 확인
   - 브랜딩 확인 (간병노트)
```

### 4️⃣ 개별 PDF 테스트
```
각 서류:
1. 미리보기
2. 다운로드
3. 디자인 확인:
   - 큰 글자 ✅
   - 깔끔한 레이아웃 ✅
   - 중앙 정렬 ✅
   - 전문적인 느낌 ✅
```
