```

2. 파일을 박스로 드래그
3. 드롭!
→ 업로드 중... ✅
→ 성공 ✅
```

### 여러 파일 업로드
```
1. "병원 영수증" → 파일 A 업로드 ✅
2. "병원 영수증" → 파일 B 업로드 ✅
```

```
2. [삭제] 클릭 → 확인 → 삭제됨 ✅
```
