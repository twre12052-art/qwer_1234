## 🔧 설정해야 할 것 (2가지)

### 필수: Supabase SQL 실행

#### 실행할 SQL (순서대로):
```
1. 0006_m4_activity_logs.sql       # Admin 활동 로그
2. 0008_m5_notification_logs.sql   # 이메일/알림 로그
```

**실행 방법**:
1. Supabase Dashboard → SQL Editor
2. 파일 내용 복사 붙여넣기
3. Run 클릭

---

### 선택: Gmail 이메일 설정

이메일 발송 기능을 사용하려면:

#### .env.local에 추가:
```bash
# Gmail 설정
GMAIL_USER=your.email@gmail.com
GMAIL_APP_PASSWORD=16자리앱비밀번호
```

#### 서버 재시작:
```
Ctrl+C → npm run dev
```
