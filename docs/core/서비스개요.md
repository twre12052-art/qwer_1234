# 📱 간병노트(가칭)

## ⭐ 1차 서비스 개요 (중도 종료 & 연장 기능 포함 최종본)

---

## 1. 서비스 소개

간병노트는 가족·지인 간병을 할 때

보험 청구에 필요한 서류를 쉽게 만들 수 있게 도와주는 앱입니다.

보호자가 간병 내용을 등록하고,

간병인은 매일 간병일지를 작성하면

필요한 모든 서류가 자동으로 PDF로 만들어집니다.

---

## 2. 사용자

- **보호자** : 간병을 맡기는 사람
- **간병인** : 간병을 직접 제공하는 사람
- **운영자** : 서류 발급 정보 제공(사업자)

---

## 3. 서비스 이용 흐름

---

## ① 보호자 회원가입

입력 내용: 이름 / 연락처 / 생년월일 / 주소

---

## ② 간병 등록

보호자가 입력하는 내용:

- 환자 이름
- 간병 장소(병원명 등)
- 간병 시작일
- 간병 종료 예정일
- 간병비(일당 또는 시급)
- 간병인 이름
- 간병인 연락처

입력이 끝나면 **간병 계약서 초안 자동 생성**.

---

## ③ 보호자 계약 동의 → 간병인에게 링크 발송

보호자가 동의하면

앱이 간병인에게 카카오톡/문자로 링크를 전송합니다.

---

## ④ 간병인 동의 절차

간병인은 아래를 입력:

- 이름
- 주민등록번호 전체
- 주소

그 후 보호자가 입력한 간병 내용을 확인하고

**“동의합니다”** 버튼을 누르면 계약이 확정됩니다.

---

## ⑤ 간병 진행

간병이 시작되면 앱은 필요할 때 카카오톡으로 안내합니다.

- 간병 시작 알림
- 매일 간병일지 작성 알림
- 간병 종료 안내

---

## ⑥ 매일 간병일지 작성(간병인)

간병일지는 매일 작성해야 합니다.

- 날짜는 앱이 자동 생성
- 내용은 간병인이 직접 체크

체크 항목 예:

식사 보조 / 체위 변경 / 투약 보조 / 이동 도움 / 배변·배뇨 도움 / 메모

모든 날짜가 작성되어야 PDF가 생성됩니다.

---

# ⭐ ⑦ 간병 기간 변경 기능 (1차에서 제공)

간병은 실제 상황에 따라 **예정보다 일찍 종료되거나**,

반대로 **더 길어질 수 있습니다.**

그래서 보호자 화면에 두 가지 버튼을 제공합니다:

---

## ✔ 1) “간병 조기 종료” 버튼 (부분취소)

예시 상황:

원래 30일까지 간병 예정이었는데

환자가 회복 또는 다른 사유로 10일 만에 종료될 경우.

### 기능 설명

- 보호자가 **간병 조기 종료** 버튼을 누르면
    
    종료일이 **현재 날짜로 변경**됩니다.
    
- 이 변경된 날짜까지만 일지가 필요합니다.
- 빈 날짜가 생기지 않도록 자동으로 조정됩니다.

### 확인 과정

- 버튼을 누르면 간단한 확인창 표시:
    
    > “간병인과 조기 종료에 대해 협의하셨나요?”
    > 
- ‘확인’ → 종료일이 즉시 변경되고
    
    간병인에게 알림이 발송됩니다.
    

---

## ✔ 2) “간병 연장” 버튼

예시 상황:

보호자가 처음엔 5일만 생각했지만

환자 상태가 좋아지지 않아 3일 더 필요할 경우.

### 기능 설명

- 보호자가 **간병 연장** 버튼을 누르고
    
    새 종료일을 선택합니다.
    
- 자동으로 연장된 날짜만큼 일지 입력칸이 추가됩니다.

### 확인 과정

- ‘연장하기’ 버튼을 누르면 아래 문구 표시:
    
    > “간병인과 연장에 대해 협의하셨나요?”
    > 
- ‘확인’ → 신규 종료일 적용
    
    간병인에게 카카오톡 알림 전송
    

---

## ⭐ 조기 종료 & 연장 기능의 공통 규칙

- 보호자만 변경할 수 있음
- 변경 시 간병인에게 카카오톡 알림 자동 발송
- 변경 후에는 **해당 날짜 기준으로 일지 작성이 필수**
- 일지가 모두 작성되기 전까지 PDF 생성 불가

---

## ⑧ 간병비 지급 정보 입력

앱이 자동 계산:

- 총 간병일수
- 총 간병비(일당 또는 시급 × 일수)

보호자가 입력:

- 실제 지급 날짜
- 지급 방식(계좌/현금)

※ 간병비는 보호자가 직접 송금합니다.

앱은 결제 기능 없음.

---

## ⑨ 보험 제출용 PDF 자동 생성

모든 작성이 끝나면 앱이 PDF를 자동 생성합니다:

- 간병 계약서
- 간병일지
- 간병인 확인서
- 소속 확인서
- 간병비 지급 확인서

서류 하단에는 고정 문구 표시:

> “이 서류는 보험 청구용이며,
> 
> 
> 연말정산 의료비 공제용 영수증이 아닙니다.”
> 

---

# 📌 요약

- 보호자: 간병 등록 → 조기 종료·연장 가능 → 지급 정보 입력
- 간병인: 본인 정보 입력 → 매일 일지 작성 → 동의
- 앱: 계약서·일지·확인서·지급확인서 PDF 자동 생성
- 매일 일지가 미작성이면 서류 생성 불가
- 조기 종료/연장 기능으로 실제 간병 상황에 맞게 기간 자동 조정
