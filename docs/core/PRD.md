PRD

---

# 1. 제품 목적

가족·지인 간병 시 보험 청구용 서류(계약서·일지·지급확인서)를  
쉽게 만들 수 있는 서비스입니다.

보호자 입력 → 간병인 확인 → 매일 일지 → 자동 PDF 생성  
이 전체 흐름을 안정적으로 구현하는 것이 1차 목표입니다.

---

# 2. 핵심 기능 (1차 제공)

- 보호자 회원가입
- 간병 등록(케이스 생성)
- 보호자 계약 동의
- 간병인 계약 동의(링크)
- 매일 간병일지 작성(간병인)
- 간병 기간 변경: **조기 종료 / 연장**
- 간병비 지급 정보 입력(보호자)
- 보험 제출용 PDF 자동 생성
- 카카오톡 알림 연동

---

# 3. 사용자 구분

- **보호자**: 간병 요청자  
- **간병인**: 간병 제공자  
- **운영자**: 사업자 정보 제공

---

# 4. 상세 요구사항

---

## 4.1 보호자 회원가입

입력:
- 이름
- 연락처
- 생년월일
- 주소

---

## 4.2 간병 등록(케이스 생성)

입력:
- 환자 이름
- 간병 장소(병원명 등)
- 간병 시작일
- 간병 종료 예정일
- 간병비(일당 또는 시급)
- 간병인 이름
- 간병인 연락처

동작:
- 입력 후 **간병 계약서 초안 자동 생성**

---

## 4.3 보호자 계약 동의

- 계약서 확인 후 "동의합니다"
- 간병인에게 전달할 **계약 링크 생성**

---

## 4.4 간병인 계약 동의 (링크 방식)

간병인은 별도 회원가입 없이 링크로 접근

입력:
- 이름
- 주민등록번호 전체
- 주소

확인:
- 계약 내용 확인 → "동의합니다"

추가:
- 보호자 화면 → **[간병인 링크 재발송]** 버튼 제공
- 간병인이 동의 완료 후 동일 링크 접속 시: “이미 동의 완료” 안내 표시

---

## 4.5 간병 진행 & 알림

카카오톡 알림:
- 간병 시작 안내
- 매일 간병일지 작성 알림
- 간병 종료 안내
- 조기 종료/연장 알림
- 일지 미작성 안내

---

## 4.6 매일 간병일지 작성

규칙:
- **매일 1개씩 작성**
- 날짜 자동 생성
- 미작성 날짜가 있으면 **PDF 생성 불가**

작성 항목:
- 식사 보조
- 체위 변경
- 투약 보조
- 이동 도움
- 배변/배뇨 도움
- 특이사항 메모

---

# 4.7 간병 기간 변경 기능 (조기 종료 / 연장)

---

## ✔ A. 조기 종료(부분 취소)

버튼: **[간병 조기 종료]**

동작:
- 종료일 → ‘현재 날짜’로 변경
- 새 종료일 이후 날짜의 일지는 **자동 비활성화**
  (PDF/화면에서 제외)

확인 문구:
> "간병인과 조기 종료에 대해 협의하셨나요?"

알림:
- 간병인에게 변경 안내 카카오톡 발송

---

## ✔ B. 간병 연장

버튼: **[간병 연장]**

동작:
- 새 종료일 선택
- 새 종료일까지 일지 입력칸 자동 생성

확인 문구:
> "간병인과 연장에 대해 협의하셨나요?"

알림:
- 간병인에게 변경 안내 발송

---

## ✔ 변경 공통 규칙

- 변경은 보호자만 가능
- 변경 후 간병인에게 알림 발송
- 변경된 날짜 모두 일지 작성해야 PDF 생성됨

---

# 4.8 간병비 지급 정보 입력 (보호자)

보호자 입력:
- 실제 지급 날짜
- 지급 방식(계좌/현금)

앱 자동 계산:
- 총 간병일수
- 총 간병비

※ 앱 내 결제 없음 (보호자가 직접 송금)

---

# 4.9 PDF 자동 생성

생성 문서:
- 간병 계약서
- 간병일지
- 간병인 확인서
- 소속 확인서
- 간병비 지급 확인서

기능:
- PDF 다운로드
- 카카오톡 전송

고정 문구:
> “이 서류는 보험 청구용이며 연말정산 의료비 공제용 영수증이 아닙니다.”

---

# 5. 케이스 상태 정의

- `보호자 동의 대기`
- `간병인 동의 대기`
- `간병 예정`
- `간병 진행중`
- `일지 작성중`
- `일지 미작성 있음`
- `서류 준비중`
- `완료(PDF 생성 완료)`

---

# 6. 1차 개발 범위 요약

- 로그인: 보호자만 필요
- 간병인: 링크만으로 접속
- 간병일지 날짜 자동 생성
- 조기 종료/연장 기능 포함
- 카카오톡 알림 연동
- PDF 생성 기능
- 결제 기능 없음
- 전자서명 없음(2차 검토 사항)
